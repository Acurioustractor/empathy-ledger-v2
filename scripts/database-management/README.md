# Database Management Scripts

This directory contains scripts for database maintenance, migrations, and data management.

## Files

- **create-galleries-tables.js** - Script to create gallery-related database tables
- **fix-params-await.js** - Fix script for async/await parameter handling
- **fix-supabase-await.js** - Fix script for Supabase async patterns
- **link-photos-to-organizations.js** - Script to link existing photos to organizations

## Usage

These scripts are typically run during development or migration phases.

**Note**: Always test these scripts in a development environment before running in production.
