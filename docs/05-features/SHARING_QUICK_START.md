## Common Share Scenarios

### ‚úÖ Share Will Work

- Story is published
- You gave explicit consent
- Story is set to public
- You're sharing to allowed platforms

### ‚ùå Share Will Fail

| Error | Why | Solution |
|-------|-----|----------|
| "Story is not published" | Story is still a draft | Publish the story first |
| "No consent for sharing" | Missing consent | Add explicit consent |
| "Privacy settings restricted" | Story is private | Change to public |

## Best Practices

### Before Sharing

1. **Read the story** - Make sure you understand it
2. **Check consent** - Verify the storyteller approved sharing
3. **Respect sensitivity** - Honor high sensitivity warnings
4. **Consider context** - Is this the right platform/audience?

### Cultural Protocols

- Always respect Elder review requirements
- Honor community-specific sharing protocols
- Get additional permission for sacred content
- Consider cultural implications of the platform

| **Email** | Email with story link | Personal sharing |
| **Embed** | HTML code for websites | Publishing content |

## Troubleshooting

### "Link copied" but nothing happens

**Problem:** Clipboard permission denied

**Solution:**
1. Allow clipboard access in browser
2. Manually copy URL from address bar
3. Try native share dialog instead

### Share count not updating

**Problem:** Browser cache

**Solution:**
1. Refresh the page
2. Clear browser cache
3. Count updates within a few seconds

### Can't share high sensitivity story

**Problem:** Missing confirmation or permissions

**Solution:**
1. Read the warning carefully
2. Confirm you have permission
3. Contact story owner if unsure
4. Consult cultural advisors

## Questions?

### Can I unshare a story?

No, but you can:
- Change privacy to private (stops new shares)
- Archive the story
- Revoke consent (disables all sharing)

### Do I get notified of shares?

Not currently, but you can:
- Check your analytics dashboard
- View share counts on stories
- See recent activity timeline

### Can I limit sharing?

Yes, through:
- Privacy settings (private = no sharing)
- Consent withdrawal (disables sharing)
- Cultural sensitivity level (adds warnings)
