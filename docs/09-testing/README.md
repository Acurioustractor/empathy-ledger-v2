## UAT Testing UI

The platform includes a built-in testing dashboard at `/admin/testing` with:

- **22 test scenarios** across 5 modules
- **Module filtering** (Syndication, Dashboard, Privacy, Knowledge Base, Onboarding)
- **Progress tracking** with completion percentages
- **Per-task feedback** (rating, difficulty, issues, comments)
- **Session management** (start, pause, reset)
- **Summary export** after completion

---

## Test Modules

### Module 1: Syndication Dashboard (6 scenarios)
Testing story sharing approval/denial workflow, revenue tracking, and distribution management.

### Module 2: Storyteller Dashboard (6 scenarios)
Testing personal dashboard navigation, transcript management, story creation, and analytics.

### Module 3: Privacy & Settings (5 scenarios)
Testing data sovereignty controls, ALMA configuration, and visibility settings.

### Module 4: Knowledge Base & Help (3 scenarios)
Testing AI-powered documentation search and help discovery.

### Module 5: First-Time User Journey (2 scenarios)
Testing onboarding flow and first story submission.
