| Token | Use |
|-------|-----|
| `bg-cream` | Backgrounds |
| `text-ink-900` | Primary text |
| `sunshine-yellow` | CTAs, accents |
| `earth-*` | Grounding elements |

## Rules

✅ Dark mode: use `dark:` variants
❌ No arbitrary colors `bg-[#xxx]`
