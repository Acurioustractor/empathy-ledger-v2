# Brand Voice & Terminology

## Respectful Terminology
| ✅ Use | ❌ Don't Use |
|--------|-------------|
| Storyteller | User, contributor |
| Story | Post, content |
| Community | Audience, followers |
| Elder (capitalized) | elder |
| Traditional territory | Location |
| Cultural background | Ethnicity |

## Standard Messaging
- **Tagline**: "Every Story Matters"
- **Primary CTA**: "Capture Your Story"
- **Secondary CTA**: "Explore Stories"
- **Footer tagline**: "Preserving Culture Through Story"
- **Contact**: hello@empathyledger.com

## Voice Attributes
- Warm, not corporate
- Respectful, not casual
- Empowering, not patronizing
- Community-focused, not transactional

## Logo Guidelines
- Placement: Top left, 40px height
- Desktop: Wordmark + icon
- Mobile (< 768px): Icon only
