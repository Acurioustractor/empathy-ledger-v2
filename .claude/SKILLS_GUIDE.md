# Claude Skills Guide

AI-Assisted Development for Empathy Ledger

## Available Skills

### Global Skills (Shared across ACT ecosystem)
- `/act-brand-alignment` - ACT brand voice and LCAA methodology
- `/ghl-crm-advisor` - GoHighLevel CRM strategy  
- `/act-github-pm` - GitHub project management expertise
- `/act-deployment-helper` - Deployment troubleshooting (coming soon)
- `/act-security-advisor` - Security best practices (coming soon)

### Local Skills (Empathy Ledger specific)
- `codebase-explorer` - Navigate Empathy Ledger architecture
- `data-analysis` - AI-powered theme/quote/story analysis
- `design-component` - Storyteller cards, story cards, UI components
- `supabase` - Database tables, relationships, query patterns
- `cultural-review` - Cultural sensitivity review
- `design-system-guardian` - Design system consistency
- `empathy-ledger-codebase` - Codebase-specific guidance
- `empathy-ledger-dev` - Development workflows
- `gdpr-compliance` - Privacy and data protection
- `story-craft` - Story creation and editing
- `supabase-connection` - Database connection helpers
- `supabase-sql-manager` - SQL query assistance
- `deployment-workflow` - Deployment processes
- `gohighlevel-oauth` - GHL OAuth integration

## Usage

### Invoke Global Skills
```
/act-brand-alignment "review this homepage copy"
/ghl-crm-advisor "design a storyteller onboarding pipeline"
/act-github-pm "create an issue for rate limiting"
```

### Invoke Local Skills
Local skills are project-specific and available in `.claude/skills/local/`
